# 🌐 Simulador Web de Crisis

Este proyecto es una aplicación web interactiva desarrollada en **HTML5, CSS3 y JavaScript**, que permite simular problemas reales relacionados con el **abastecimiento, precios y consumo familiar en contexto de crisis**.  

El objetivo es mostrar, mediante modelos matemáticos sencillos, cómo factores como el aumento de precios, el transporte con desvíos o la pérdida del poder adquisitivo afectan directamente a las familias.

---

## 🚀 Características principales

- Estructura semántica con etiquetas **HTML5** (header, nav, main, section, footer).
- Estilos aplicados con **CSS externo**, paleta de colores coherente y diseño responsivo.
- Uso de **JavaScript y DOM** para cálculos dinámicos y visualización de resultados.
- Formularios interactivos para ingresar datos y obtener simulaciones.
- Resultados mostrados en pantalla mediante tarjetas y mensajes claros.
- Casos de estudio incluidos para comprobar el funcionamiento.
- Proyecto organizado en carpetas (HTML, CSS, JS, imágenes).
- Publicación en línea mediante **GitHub Pages**.

---

## 📊 Escenarios simulados

- **Precios de alimentos:** cálculo del gasto familiar antes y después de un aumento de precios.  
- **Transporte con desvío:** estimación del gasto adicional por rutas más largas.  
- **Presupuesto familiar:** verificación de si el presupuesto alcanza para cubrir compras básicas.  

---

## 📦 Cómo ejecutar

1. Clonar el repositorio.
2. Abrir `index.html` en un navegador.
3. Ingresar datos en los formularios y presionar **Calcular**.
4. Revisar los resultados dinámicos en pantalla.

---

## 📑 Casos de estudio

Ejemplo de prueba incluido en la página:  
- Producto: Arroz  
- Precio anterior: 8 Bs  
- Precio actual: 11 Bs  
- Cantidad mensual: 10 unidades  

Resultado esperado: cálculo del gasto anterior, gasto actual y diferencia.
